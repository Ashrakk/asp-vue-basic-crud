# ASP.NET Core 8.0 & Vue 3 Basic Crud Application 

Created to learn about ASP and Quasar framework

## Features
- Table viewer
- server side filters
- server side virtual infinite pagination
- add record
- update record
- delete record

## Build and run
```
cd Frontend
npm install
npm run dev
```

```
cd Backend/CorsoASP.UI
dotnet run
```
